## Color Palette

### Primary Colors
- **Purple/Indigo**: #6A409A, #8B5CF6, #C084FC
- **Blue**: #3B82F6
- **Magenta**: #EC4899, #E879F9
- **Neutrals**: #F8FAFC, #E2E8F0, #CBD5E1, #64748B, #374151

### Gradient Combinations
1. **X402 Gradient**: Blue → Purple → Magenta
2. **Workforce Text**: Light gray gradient untuk readability
3. **Tech Elements**: Purple variants untuk consistency
4. **Corporate**: Dark grays untuk profesional look

## Brand Identity Guidelines

### Typography
- **Primary Font**: Arial, sans-serif
- **Weight**: Bold untuk brand name, Regular untuk tagline
- **Letter Spacing**: 0.5px - 2px tergantung design
- **Hierarchy**: Clear differentiation antara "WORKFORCE" dan "X402"

### Visual Elements
- **X402 Integration**: Selalu prominent di setiap logo
- **Tech Symbolism**: Circuit patterns, connectivity, AI elements
- **Professional Appeal**: Corporate colors untuk business contexts
- **AI Workforce Theme**: Human-tech integration

### Usage Guidelines
- **Preferred Format**: SVG untuk scalability
- **Minimum Size**: 120px width untuk readability
- **Clear Space**: Maintain 25% padding around logo
- **Background**: Works on light and dark backgrounds
- **Consistency**: Gunakan warna gradient yang konsisten

## Technical Specifications

### File Formats
- **SVG**: Primary format (scalable, editable)
- **PNG**: Untuk web use (can be exported from SVG)
- **ICO**: Untuk favicons (using Icon.svg)

### Responsive Design
- **Desktop**: Full logo dengan semua elements
- **Mobile**: Consider using Icon.svg atau minimal version
- **Print**: SVG can be exported to high-res PNG/PDF

## Logo Selection Guide

| Context | Recommended Logo | Reason |
|---------|------------------|---------|
| Website Header | Concept Logo | Most detailed, complete brand story |
| Mobile App | Icon Logo | Square format, recognizable |
| Business Cards | Corporate Logo | Professional, clean |
| Social Media | Icon Logo | Square format, compact |
| Technical Docs | Tech Logo | Matches tech aesthetic |
| Legal Documents | Corporate Logo | Formal appearance |
| Presentations | Corporate/Concept | Depending on audience |
| API Documentation | Tech Logo | Developer-focused |
